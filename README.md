# Employee Management System (PHP + MySQL)

A simple and clean Employee Management System built using PHP, MySQL, and HTML/CSS.  
It performs complete CRUD operations:

- Add Employee  
- View Employee Details  
- Edit Employee Information  
- Delete Employee  
- Display Employees in a Table  

## Features

✔ Add new employees  
✔ View full employee details  
✔ Update employee records  
✔ Delete employees
